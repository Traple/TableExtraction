Computer simulations of reactions were carried out using the computer program ChemSim which was placed at our disposal by R. Kramer prior to publication (the program used in this version is available on request from Kramer & Hofmann, Datenbankdesign Hard- and Software, Saar- briicken-Giidingen, F RG). The program simulates chemical reactions by numerical integration of the underlying differen- tial equations system. Integration intervals are determined by a ﬂux-tolerance method [36]. It allows the simulation of combined rate equations and equilibria. Integrations are cal- culated by a second-order Runge-Kutta procedure. For calcu- lation of coupled equilibria, a Newton’s zero search in several dimensions is used. Therefore, equations are generated which include the laws of mass action, the ion product of water (if necessary) and the equations necessary to meet the law of constant mass. The number of equations has to coincide with the number of species in the simulated system. For all equations, the program determines the analytical derivatives which are used to solve the zeros problem rapidly. The pro- gram was run on a Atari 1040 ST +.
