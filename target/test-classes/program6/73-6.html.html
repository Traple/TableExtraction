This would also support the hypothesis that AdhF and Adh” enzymes originally come from the genes for the Adh‘ enzyme, because there is an amino acid substitution (Lys —>Thr) in this peptide in both allelic variants (Thatcher, 1980).

Peptides TII3 and TIl speciﬁc for arginine are common to all three species, and peptide TIV3 only appears with the D. simulans and D. melanogaster enzymes, being absent with the D. virilis enzyme. The amino acid analysis of this peptide after elution shows that it only contains arginine.
