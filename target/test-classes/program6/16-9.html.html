Structure determination and refinement

Three zinc sites were identified with an anomalous Patterson search implemented in SHELX-90:°’2 Two of these sites were consistent with the 222 point group molecular symmetry of A. pernix ADH, which allowed determination of the NCS symmetry operations. These sites correspond to the catalytic zinc ions of the enzyme. A third strong zinc site was located on the interface between tetramers, and was presumed to be an artefact of crystallisation. Refinement of the zinc ions positions, occupancies and anomalous occupancioes was conducted using MLPHARE.” Phasing to 1.88 A resolution was performed using the same program. The resulting Figure of merit of the phases was 0.284. The structural zinc ion positions were identified using difference anomalous Fourier synthesis with the MLPHARE phases. However, due to their partial occupancies, the structural zinc ions were not used in the phase calculations.

Improvement of the phases was achieved by 2-fold NCS averaging implemented in the program DM.“ Phased molecular replacement, based on the spherically averaged phased translation function35 was performed by MOLREP.36 Human ADH B1 isozyme37 (PDB code 1HSZ) was positioned into the averaged map to create an initial model. The A. pernix ADH sequence was then built into the model using the program O38 Refinement was performed using the program REFMAC516 (Table 1). The cofactor NADH was added using the REFMAC5 dictionary for this molecule. Therefore, the nomenclature of NADH used here is that of REFMAC5. Water molecules were added using the program WODA (A. Teplyakov, personal communication), refined and checked manually using the 2P0 — PC map.

The inhibitor was built in the Sketcher interface of the program LIBCHECK” and positioned into the additional density of the active site using O. Further refinement was performed by REFMAC5 and the structure of the inhibi- tor modified to give the best fit to both the 2P0 — PC and PO — PC maps. The inhibitor model was optimised by ensuring that the B-factors of the atoms were consistent both within the molecule and with the B-factors of neigh- bouring amino acid residues. The two terminal atoms of the inhibitor interact with the catalytic zinc ion and the hydrogen bond-forming residues. When either of these atoms was described as nitrogen, the B-factor value became unreasonably low, suggesting that both were oxygen atoms and that the inhibitor had a terminal car- boxylate group. This approach produced the molecule octanoic acid as the most accurate description of the inhibitor density.

Ionic bonds, hydrogen bonds and hydrophobic inter- actions were assigned manually in the oprogram O, using bond length cut-offs of 3.0 A, 3.2A and 4.5 A, respectively. Superimpositions of ADH structures were performed using O. These were subsequently subjected to minimisation in O, using default program values, to obtain optimal superimposition for a given zone.

Protein Data Bank accession code

The structure factors and refined coordinates of the A. pernix ADH have been deposited with the Protein Data Bank; the accession code is 1H2B.
